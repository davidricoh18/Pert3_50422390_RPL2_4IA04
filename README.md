Pada praktikum RPL 2 pertemuan 1, saya mempelajari arsitektur Model–View–Controller (MVC), yaitu pola pemisahan kode menjadi tiga komponen utama. 
Model bertanggung jawab menyimpan dan mengelola data, View menangani tampilan antarmuka kepada pengguna, sedangkan Controller berfungsi sebagai penghubung yang menerima input pengguna dan mengatur alur data antara Model dan View. 
Dengan pemisahan ini, aplikasi menjadi lebih terstruktur, mudah dikembangkan, dan lebih efisien ketika dikerjakan secara tim.
